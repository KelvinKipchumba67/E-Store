## About

E-Store is a lightweight front-end e-commerce demo that showcases product listing, shopping cart interactions, and responsive UI built with MERN. It is intended as a starting point for learning and prototyping store-like interfaces.

## Features

- Product listing and search (UI)
- Product detail view
- Client-side cart with add/remove/update quantity
- Responsive layout for desktop and mobile
- Simple, easy-to-read code suitable for learning and customization

## Tech stack
MERN
